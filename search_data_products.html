Welcome to the Data Mesh Query Tool

This tool allows you to ask questions and retrieve insights from various data products within the AWS Data Mesh ecosystem, including services like Sales Order, Patient Service, Activity Service, UPP, LIMS, and Billing Eligibility Service.

How to Use

  1. Enter your question in plain text in the input box.
  2. Click on the "Search" button to query your data products.
  3. Analyze the results to understand data gaps and opportunities.
